# Vein Project

## Overview
Basic Node.js/Express web application imported from GitHub repository: https://github.com/uforgotmewhy1/vein

The repository was empty at import time, so a starter web application was created.

## Project Architecture
- **Backend**: Node.js with Express (server.js)
- **Frontend**: Static HTML/CSS served from public/ directory
- **Port**: 5000 (configured for Replit environment)

## Recent Changes
- 2025-11-16: Initial setup of Node.js/Express application
- Created basic frontend with HTML/CSS
- Configured server to bind to 0.0.0.0:5000 for Replit compatibility

## Development
- Run command: `npm start`
- Server runs on port 5000
- Static files served from `public/` directory
