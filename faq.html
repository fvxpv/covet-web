Frequently Asked Questions

What is Covet Bot?
Covet is a multipurpose Discord bot designed to enhance your server with moderation, automation, utility features, giveaways, tickets, starboard, music, and more. It's a comprehensive solution for managing and engaging your community.
How do I invite Covet to my server?
You can invite Covet using the invite link: click here. Make sure you have "Manage Server" permissions to add bots.
What is the default prefix for Covet?
The default prefix is . (period). You can customize it per server using the .prefix set <new_prefix> command.
What permissions does Covet need?
Covet requires Administrator permissions to function properly. This allows it to manage channels, roles, messages, members, and other server features without restrictions.
How do I set up moderation logging?
Use .modlog setup to initialize the moderation logging system. Then use .modlog channel set #channel-name to designate where mod actions will be logged.
Can I customize Covet's appearance for my server?
Yes! Server owners can customize the bot's avatar and banner using .customize avatar <url> and .customize banner <url> commands.
How do I create a giveaway?
Use .giveaway start <duration> <winners> <prize> to create a giveaway. For example: .giveaway start 1h 1 Discord Nitro. Members can react to enter!
Does Covet have music features?
Yes, Covet includes music playback features. You can play songs, create queues, and control playback in voice channels.
How do I get support or report bugs?
Join our support server at https://discord.gg/5ry3HysZ7E for help, feature requests, and bug reports. Our community is ready to assist you!
Is Covet free to use?
Yes, Covet is completely free to use with all features available to every server. We're committed to providing a powerful bot accessible to all Discord communities.
